This contains the set of instructions required to build and run arquillian sample.

Building the Project.
====================

Pre-requisites
--------------
JDK 1.6.x or higher
Maven 2 or higher
java_home and m2_home has been set.

Steps:
------
1 - download the project.
2 - Go to the project home directory and run 
    $ mvn clean test -Parq-jboss_as_managed_7.x
